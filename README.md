# VOW SEOUL 모바일 청첩장 제작 플랫폼 (MWI Ver 2.0)

VOW SEOUL 모바일 청첩장 제작 플랫폼은 프리미엄 지류 청첩장의 브랜드 아이덴티티와 편의성을 연동하여, 고객 정보 수집(폼 빌더)부터 모바일 청첩장 자동 생성 및 커스텀 편집기, 그리고 하객 RSVP 및 방문 통계 대시보드에 이르는 전 과정을 자동화하는 프리미엄 통합 시스템입니다.

---

## 1. 프로젝트 목적 (Purpose)

* **제작 공정의 자동화 및 오류 최소화**: 기존 구글 폼 및 수작업 제작으로 인해 발생하는 시간 지연과 오타 등의 인적 오류를 최소화합니다.
* **지류와 모바일의 결합**: 지류 청첩장의 라인업별 필요 사양을 폼 빌더와 매핑하여, 필요한 정보를 누락 없이 일목요연하게 수집합니다.
* **디자인 및 레이아웃의 유연한 관리**: 피그마 등 외부 디자인 툴로 제작한 HTML/CSS 파일을 업로드하고, 특정 데이터 필드와 연동하여 코드 수정 없이 실시간 편집 및 테마 교체를 지원합니다.
* **고객 편의성 극대화**: 예식 정보 수정, 하객 RSVP 확인, 방명록 관리, 트래픽 분석 등의 관리 기능을 고객에게 전용 대시보드로 실시간 제공합니다.

---

## 2. 사용자 역할 (Target Users)

1. **플랫폼 어드민 운영자 (VOW SEOUL Administrator)**
   * 고객 정보 및 진행 현황 모니터링
   * 지류 청첩장 종류별 수집 폼 생성, 필드 라이브러리 설정 및 버전 관리
   * 고객이 제출한 내용 조회, 강제 수정, 고화질 원본 이미지 다운로드 및 청첩장 초안 수동 업데이트
   * 매출 통계, 테마 선호도, BGM 순위 및 전체 시스템 트래픽 분석
2. **테마 디자이너 (Theme Designer)**
   * 피그마 디자인 기반 HTML/CSS 파일 업로드 및 테마 등록
   * 데이터 필드와 테마 내 요소 바인딩 규칙(디자인 토큰) 설정
   * 색상 팔레트, 폰트 프리셋, 레이아웃 변경 범위 지정
3. **결혼 고객 (Couples / 신랑 신부)**
   * 카카오톡 인-앱 브라우저 등 모바일 환경에 최적화된 폼 입력 페이지를 통해 예식 정보 작성 (임시저장 제공)
   * 모바일 청첩장 발행 후 하객 RSVP 현황, 방명록 공개/비공개 처리 및 누적 방문자 수 통계를 조회할 수 있는 전용 대시보드 이용
4. **하객 (Guests)**
   * 발행된 모바일 청첩장 주소를 통해 식장 정보, 갤러리 슬라이드, 축의 계좌 복사, 지도 검색, 배경음악 청취, RSVP 참석 응답 및 방명록 작성 기능 이용

---

## 3. 핵심 기능 (Core Features)

### 3.1 맞춤형 정보 수집 폼 빌더 (Form Builder)
* **드래그 앤 드롭 빌더**: 신랑/신부, 예식 정보, 혼주, 송금 계좌 등 필요한 입력 양식을 자유롭게 구성.
* **SELECT_TEXT 카드형 셀렉터**: 콤마로 구분된 긴 항목 대신 개별 카드별 다중 선택지를 입력하고 직접 입력(`✏️ 직접 입력하기`)을 포함할 수 있는 세련된 UI 제공.
* **timentext (식순 타임라인) 필드**: 결혼식 당일 시간별 세부 식순(`[12:00 | 입장]`, `[12:10 | 주례]`)을 추가하고 드래그 삭제 및 동적 행 생성 가능.
* **더블 레이어 임시저장**: 카카오톡 인-앱 브라우저 등 휘발성이 높은 모바일 환경에서 입력 도중 유실을 막기 위해 입력할 때마다 `localStorage`에 자동 백업(`vowseoul_draft_{slug}`)하고, 저장 버튼 클릭 시 Supabase DB에 최종 반영.

### 3.2 청첩장 초안 자동 생성 및 3단 패널 편집기 (Editor)
* **자동 초안 매핑**: 고객이 폼 제출을 완료하면 해당 데이터를 바탕으로 1초 만에 텍스트, 갤러리 이미지, 지도 좌표, 음악을 적용한 초안 발행.
* **실시간 3단 프리뷰 에디터**: 텍스트 인라인 수정, 갤러리 크롭 및 10대 테마 블록 순서 조정(Drag & Drop)에 따른 실시간 모바일 프리뷰 화면 제공.

### 3.3 고객 전용 실시간 관리 대시보드
* **실시간 참석(RSVP) 분석**: 하객이 제출한 참석 여부, 셔틀 탑승률, 식사 여부를 실시간 도넛 차트로 집계 및 엑셀 다운로드 제공.
* **방명록 유해물 관리**: 축하 메시지 목록 중 무단 광고나 원치 않는 방명록을 비공개로 토글 처리.
* **개인정보 자동 파기**: 보안 유지를 위해 예식일 14일 이후 하객 RSVP 개인정보 데이터를 안전하게 영구 파기.

### 3.4 어드민 관리 포털 & 데이터 분석 대시보드
* **실제 데이터 기반 대시보드**: Mock/더미 데이터 대신 Supabase 실데이터(`customers` & `invitations`)와 조인하여 실시간 결제 건수, 당일 매출(건당 5만원), 예정 예식 수 집계 및 주간 매출 라인차트 표기.
* **테마/BGM 선호도 통계**: 데이터베이스 내 청첩장 적용 테마 버전 및 사용 음악 ID를 카운트하여 비중 및 사용 순위를 차트로 시각화.
* **제출 응답 상세 뷰어**: 일반 화면에서는 오타 방지 및 쉬운 텍스트 복사를 위해 드래그 선택이 기본인 읽기 전용 상태를 유지하며, 수정 버튼 클릭 시에만 입력 필드가 노출되도록 제어. 또한 첨부 이미지는 손실 없는 원본 Blob 다운로드 지원.
